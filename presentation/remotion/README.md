# TernRoute Remotion presentation

A React/Remotion presentation of the router implementation: zero-token classification, exact allowlist selection, deterministic Fireworks dispatch, bounded concurrency, validation-driven rerouting, and telemetry.

## Run

```bash
cd presentation/remotion
npm install
npm run studio
```

## Verify and render

```bash
npm run check
npm run render
```

The 1920×1080, 30 fps video is written to `out/ternroute-modern.mp4`. Add narration using [`TRANSCRIPT.md`](TRANSCRIPT.md).
